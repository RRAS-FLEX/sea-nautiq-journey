# Agency 30-Day Execution Board

## Week 1 (Days 1-7): Trust + Baseline
- [ ] Apply trust SQL hardening in prod.
- [ ] Validate overlap prevention with concurrent booking simulation.
- [ ] Run backup/restore test.
- [ ] Establish P1/P2 alert channels and owner rotation.

## Week 2 (Days 8-14): Compliance + Controls
- [ ] Finalize data retention rules.
- [ ] Enable DSR ticket workflow with evidence log.
- [ ] Validate policy-to-product alignment (privacy/refund/terms).

## Week 3 (Days 15-21): Reliability + Ops
- [ ] Add uptime checks and error dashboards.
- [ ] Run game-day drill: payment outage.
- [ ] Run game-day drill: notification queue delay.

## Week 4 (Days 22-30): Finance + Go-Live Confidence
- [ ] Daily reconciliation process running.
- [ ] Weekly variance report accepted.
- [ ] P1 incident simulation completed with documented response under SLA.
- [ ] Go-live readiness review signed.

## Hard Exit Criteria
- Booking success >= 98%
- Overlap confirmations = 0
- Reconciliation variance < 1%
- P1 response < 15 minutes
